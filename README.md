# DigiSone
Content-Based Medical Image Retrieval for Dermatological Disease Diagnosis Using Deep Metric Learning
**Dr. Abbas Fadhil Jasim AL-Gburi**
Iraqi General Customs Authority (IGCA), Baghdad, Iraq
## Results
| Metric | Value |
|--------|-------|
| mAP | 0.8452 |
| P@5 | 0.8659 |
| nDCG@10 | 0.8665 |
| SARR | 0.9054 |
## Dataset
HAM10000 — 10,015 dermoscopic images, 7 classes
